# Dijkstra Algorithm in Python

This project implements Dijkstra's algorithm to find the shortest path in a graph using Python.

## Features
- Uses priority queue (heapq)
- Efficient shortest path calculation

## Example
Graph:
A -> B (1), C (4)  
B -> C (2), D (5)  
C -> D (1)

Output:
{'A': 0, 'B': 1, 'C': 3, 'D': 4}

## How to run
```bash
python main.py
```
